Introdução Conceitos Naturais

Definindo os naturais:

Seja \(\mathbb{N}\), o conjunto dos números naturais, definido como o conjunto de elementos infinitos assegurado pelo axioma do infinito.

Sejam \(0,1,2,...\) representações de elementos de \(\mathbb{N}\) tais que: \begin{gathered}0:= \emptyset; \qquad 1:= 0 \cup \{0\}; \\ \qquad 2:= 1\cup \{1\}; \qquad...\end{gathered}

Soma de naturais:

Define-se a função \(+:\mathbb{N} \times \mathbb{N} \rightarrow \mathbb{N}\)
