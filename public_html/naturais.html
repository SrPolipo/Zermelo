Introdução Conceitos Naturais

Definindo os naturais:

Seja \(\mathbb{N}\) definido como o conjunto de elementos infinitos assegurado pelo axioma do infinito.

Sejam \(0,1,2,...\) representações de elementos de \(\mathbb{N}\) tais que: \[0:= \emptyset; \qquad 1:= 0 \cup \{0\}; \qquad 2:= 1\cup \{1\}; \qquad...\]
